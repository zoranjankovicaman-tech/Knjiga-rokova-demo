🔒 Firebase Security Rules

Postavite ova pravila u Firebase Console → Realtime Database → Rules. Štite strukturu podataka od neispravnih upisa.

{ "rules": { "terrains": { ".read": true, "$id": { ".write": true, ".validate": "newData.hasChildren(['name','pin','order'])" } }, "storeConfig": { ".read": true, "$id": { ".write": true, ".validate": "newData.hasChildren(['name','email','pin','terrainId'])" } }, "stores": { ".read": true, "$storeId": { "items": { "$itemId": { ".write": true, ".validate": "newData.hasChildren(['barcode','expiryDate'])" } } } } } }
